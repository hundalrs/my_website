Projects

Hired Dev

HiredDev is an ongoing project that I am working on, and it is one that I am personally using on my job search. The frontend is built using React/Redux to aid in predictable state management, and the backend is built using Express, Node, and MongoDB/Mongoose. The main purpose of this project is to scale out this application as much as possible and in the end, give users a cheap option to track their application process. I am still going through iterations of this project before I start taking on users, but recently, I have set up a CI/CD pipeline using Github, Travis CI, Docker Hub, and AWS Beanstalk. Ultimately, this has allowed me to streamline my application development and build at a production level.

HBnB

HBnB is an AirBnB clone that I created using mostly Python and various technologies such as Flask, MySQL, and SQLAlchemy. The final product of this project was an application I created from scratch to deployment (currently not hosted). The main focus for this project was to create a robust backend opposed to a beautiful frontend. Coming out of this project, I feel very comfortable with MySQL, SQLAlchemy and SQL just in general, Flask, and Python. Deployment was especially rewarding because this was my first deployed application and because I deployed with a load balancer, web server, and firewall. Moving forward, the deployment process gave me an idea of how I would scale bigger applications.

Simple Shell

The Simple Shell is a low-level project in C in which I wrote a Unix Command line interpreter using various system calls. In this project, I handled basic POSIX commands like ls and pwd, and I also handled the PATH so the exact path of these commands do not have to be specified. This was one of my favorite projects because it truly made me understand the internals of the shell and gave me a deeper understanding of what is happening at the low-level.
